# Roman Reels — 6 реальных кейсов для промо-видео

**Дата:** 2026-04-20
**Автор:** Дмитрий Волков + Claude
**Статус:** утверждён пользователем, готов к реализации

## 1. Цель

Снять вертикальный Reels / Shorts **67 секунд** для my.linkeon.io, показывающий личного ассистента Романа на 6 реальных пользовательских задачах. Формат — «без сторителлинга, только до/после». Каждый кейс ≈ 10 сек: один короткий промпт → один конкретный артефакт.

**Целевая аудитория:** SMB-предприниматели + selfmade-профессионалы (фрилансеры, консультанты, эксперты). Общий знаменатель — «тот, кто иначе нанял бы джуна-ассистента».

**Целевое действие зрителя:** перейти на my.linkeon.io и зарегистрироваться (стартовый бонус — **25 000 токенов в подарок**).

## 2. Технический контекст

- Роман = Universal Agent. Проксируется из бэкенда `/chat` → `https://r.linkeon.io/chat` (Claude Code tool-using agent). См. `spirits_back/src/chat/chat.service.ts:75-78` и `streamUniversalAgent` метод.
- У Романа есть доступ к bash / файловой системе / WebFetch / WebSearch / image-gen. Может создавать выходные файлы (HTML, .docx, .xlsx, .csv, .pdf, .png) и возвращать их ссылками в чате.
- Фронт my.linkeon.io показывает эти ссылки как кликабельные (см. ChatInterface.tsx).
- Мобильный viewport 390×844 — основной для Reels.

## 3. Финальные 6 кейсов (порядок в видео)

### Кейс 1 — Лендинг йога-студии (#10)
- **Промпт:** «Сделай лендинг одной страницей для моей йога-студии. Название — "Дыхание". Оффер: 10 занятий за 4 900 ₽ вместо 7 000 ₽ для новичков. Добавь форму записи, расписание на неделю и кнопку "Записаться". Минимализм, пастельные тона. Сохрани как index.html»
- **Артефакт:** HTML-файл, открывается в мобильном Safari на весь экран
- **Риск:** низкий. HTML — plain text, Роман точно сделает.

### Кейс 2 — 5 постов Telegram (#6)
- **Промпт:** «Напиши 5 постов для Telegram-канала про запуск онлайн-курса "3 месяца — новый финплан". Аудитория — молодые родители 25–35. Форматы: анонс, бэкстейдж, цитата-провокация, отзыв, призыв. По 500–800 знаков, эмодзи»
- **Артефакт:** 5 постов прямо в чате Романа, показываем прокруткой
- **Риск:** низкий. Чистый текст.

### Кейс 3 — Договор с самозанятым (#2)
- **Промпт:** «Составь договор оказания услуг с самозанятым на 80 000 ₽/мес. Заказчик — ИП Волков Д.В., ИНН 770100123456. Исполнитель — самозанятый Иванов И.И., ИНН 770200654321. Услуги — SMM-ведение Instagram, 3 мес. Сохрани как .docx»
- **Артефакт:** .docx открыт в Word / Pages / Google Docs на десктопе
- **Риск:** средний. Требует python-docx на агент-сервере. Если не получится — fallback на .txt/.pdf с тем же содержанием.

### Кейс 4 — Прайс → Excel с переводом (#5)
- **Промпт:** Загружаем фото/PDF с прайсом (см. §6 — фейковый прайс парикмахерской, готовит Claude) → «Переведи прайс на английский, оформи в Excel две колонки RU/EN, сохрани как price_en.xlsx»
- **Артефакт:** .xlsx открыт в Numbers / Excel
- **Риск:** средний. Требует openpyxl. Fallback — CSV.

### Кейс 5 — 10 компаний лазерной резки → CSV (#8)
- **Промпт:** «Найди 10 компаний в Москве, которые делают лазерную резку металла. Собери название, сайт, телефон, район. Оформи как CSV»
- **Артефакт:** CSV в таблице (Numbers / Excel), подсветка «10 контактов за ~30 сек»
- **Риск:** средний. Точность телефонов зависит от веба. Если часть не находится — допустимо «8 из 10», при условии что каждая запись выглядит реальной.

### Кейс 6 — PDF-отчёт по конкурентам (#12)
- **Промпт:** «Собери отчёт по конкурентам в нише "пилатес-студия Санкт-Петербург". Найди 10 студий: название, район, средний чек, оценку, число отзывов. Сравни в таблице, построй график цен. Сохрани как PDF»
- **Артефакт:** PDF с таблицей и графиком на весь экран
- **Риск:** высокий. Длинная цепочка: web search → сбор данных → matplotlib → reportlab/weasyprint. Fallback сценарии перечислены в §6.

## 5. Структура съёмки одного кейса

Снимаем **screen recording 12–15 сек с реального iPhone** (через QuickTime на Mac) — в монтаже режем до 10 сек. Три логические части:

1. **0–2 сек:** мобилка, чат с Романом, промпт уже вставлен в поле ввода (не печатаем вживую)
2. **2–8 сек:** нажатие «Отправить» → стриминг ответа Романа → появляется ссылка на файл
3. **8–12 сек:** тап по ссылке → артефакт на весь экран

**Параллельно — статика артефакта на десктопе:** открытый PDF / HTML / .xlsx / .docx без мобильной рамки, во весь экран. Для монтажных врезок, где зритель видит чистый результат.

## 7. Risks и план B

| Риск | Вероятность | Mitigation |
|---|---|---|
| Роман не отдаёт .docx | средняя | Fallback на .pdf или .txt с тем же содержанием |
| Роман не отдаёт .xlsx | средняя | Fallback на .csv |
| Роман не собирает PDF-отчёт с графиком | высокая | Fallback: «таблица+график в чате» или замена кейсом #7 (unit-экономика) |
| Телефоны компаний некорректны | средняя | Допустимо 8/10 реальных записей; явно некорректные — редактируем руками перед съёмкой |
| Сессии Романа на r.linkeon.io падают | низкая | Ретрай через 10 сек, в худшем случае — перенос съёмки на другой день |
| Генерация слишком долгая (> 60 сек) | средняя | Ускоряем в монтаже 2–3×, делаем cross-dissolve |
